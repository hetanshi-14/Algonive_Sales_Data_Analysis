# Algonive_Sales_Data_Analysis (Task-2)
Sales Data Analysis using Python and visualization

## 📌 Project Overview
This project focuses on analyzing sales data to extract meaningful insights and identify business trends. The goal is to understand sales performance, customer behavior, and profitability using data analysis techniques.

---

## 🎯 Objective
- Analyze sales data to identify trends and patterns  
- Understand regional and category-wise performance  
- Discover profit driving factors  
- Support data driven decision making  

---

## 📂 Dataset
- **Dataset Name:** Superstore Dataset  
- Contains information about:
  - Orders
  - Sales
  - Profit
  - Customers
  - Regions

---

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔍 Steps Performed

### 1. Data Cleaning
- Removed missing values  
- Renamed columns  
- Checked data types  

### 2. Exploratory Data Analysis (EDA)
- Analyzed sales distribution  
- Studied category & sub-category performance  
- Examined regional trends  

### 3. Data Visualization
- Bar charts  
- Line graphs  
- Heatmaps  
- Pie charts  

---

## 📊 Key Insights
- Certain categories generate high sales but low profit  
- Some regions perform better than others  
- Discount impacts profit negatively in many cases  

---

## 📈 Conclusion
This analysis helps in understanding business performance and identifying areas of improvement. It provides valuable insights that can help in increasing revenue and optimizing sales strategies.
